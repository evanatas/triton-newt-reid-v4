# 🦎 triton-newt-reid-v4 — повторная идентификация тритонов по узору брюшка

Компьютерное зрение для **re-identification** (повторного опознания) тритонов по индивидуальному узору
тёмных пятен на брюшной стороне тела. Задача — узнать особь при **повторном отлове** спустя недели/месяцы,
когда узор деформируется по мере роста животного (temporal re-ID).

**Главный результат** (на запечатанном held-out наборе, тритон Карелина):

| Метрика | Значение |
|---|---|
| **top-1 (rolling)** | **0,79** [довер. интервал Уилсона 0,70–0,86] |
| top-1 (один референс) | 0,70 |
| Recall@5 / Recall@10 | 0,92 / 0,93 |
| Локальные vs глобальные признаки | SIFT **0,79** ≫ MegaDescriptor 0,30 ≫ DINOv2 0,17 (критерий Макнемара p < 0,001) |
| Прирост над прежней версией | **×4,2** (было 0,188) |
| Задержка обработки фото | медиана **1,9 с** (MPS/M2 Pro) |

Ключевая идея: прежняя версия делала ставку на **глобальный нейросетевой эмбеддинг** и упёрлась в 0,188.
Здесь — разворот на **сопоставление локальных признаков** (SIFT + аффинная геометрическая верификация +
накопление галереи), что и дало прирост в 4,2 раза на тех же данных.

## 🚀 Быстрый старт (3 команды)

Демо работает **сразу после клонирования** — каталог известных особей уже лежит в репозитории.

```bash
git clone https://github.com/evanatas/triton-newt-reid-v4.git
cd triton-newt-reid-v4
pip install -r requirements.txt
streamlit run backend/demo.py
```

Откроется `http://localhost:8501` — вкладка **«Опознание»**: загрузите фото брюшка (примеры — в `examples/`)
или выберите готовую пробу, получите **top-K похожих особей** карточками с «уверенностью %», вердикт
**известная / на проверку / новая** и наглядный **оверлей совпавших пятен**.

> При первом опознании загруженного фото один раз скачивается модель сегментации BiRefNet с HuggingFace
> (нужен интернет); дальше всё работает офлайн. Выбор готовой пробы работает без сегментации и без сети.

## 🧠 Как это работает

Конвейер обработки одного фото:

```
фото брюшка
  → сегментация тела нейросетью BiRefNet
  → маскирование бумажной метки-номера (анти-утечка идентичности)
  → тугой кроп + нормализация масштаба (холст 512×512, высота маски 440 px)
  → перевод в серый + CLAHE (порог 2,0)
  → SIFT (1500 точек) + фильтр Лоу (0,75)
  → аффинная верификация RANSAC (estimateAffine2D, порог 5 px) → число инлайеров = сходство
  → накопление галереи (rolling): максимум по прежним снимкам особи
  → вердикт: известная / на проверку / новая (калиброванный порог)
```

**Почему локальные признаки, а не эмбеддинг.** Глобальный дескриптор кодирует всё изображение и цепляется
за форму тела/позу/условия съёмки, а не за сам узор. Локальное сопоставление опирается на геометрически
согласованные соответствия отдельных пятен — именно то, что различает особей. У истинных пар — медиана
20 совпавших пятен, у разных особей — 3–4.

## 🔬 Воспроизведение результатов

Демо и метрики работают на уже готовых артефактах в репозитории. Полный пересчёт **с сырых фото** требует
данных когорт (публично не распространяются; предоставляются ИПЭЭ РАН по запросу для исследовательских
целей) и переменной `TRITON_ROOT`:

```bash
export TRITON_ROOT=/путь/к/скачанным/данным      # Windows: $env:TRITON_ROOT="C:\путь\к\данным"

cd registry && python build_registry.py && python build_splits.py   # реестр + сплиты
cd ../segment && python build_crops.py                              # кропы всех когорт
cd ../core   && python eval_core.py --cohort TK                     # оценка на dev (ожидается 0,629 / 0,753)
```

Всё детерминировано: фиксированный `seed = 42`, повторный прогон даёт идентичные метрики.
⚠️ Запечатанный тест (`registry/sealed_manifest.json`) вскрывается **один раз** и уже вскрыт — числа 0,79
заморожены и не пересчитываются.

## ⚠️ Честные оговорки

- **top-1 = 0,79 — это режим накопления галереи** (rolling); по единственному прежнему снимку — 0,70.
- Итоговая оценка — на **целевом виде** (тритон Карелина); ребристый тритон снят в облегчённом протоколе.
- **Планка top-5 ≥ 0,95 не достигнута** (получено 0,92) — подаётся честно.
- Оценка открытого множества (BAKS 0,67 / BAUS 1,0 / G 0,819) — **индикативная**, на малой выборке (17 новых особей).
