# Данные: происхождение и условия использования

Проект использует четыре когорты данных. Условия по каждой — ниже.

## Целевые данные (тритон Карелина TK, ребристый тритон PW, лабораторная когорта LAB)

- **Источник:** Институт проблем экологии и эволюции им. А. Н. Северцова РАН (ИПЭЭ РАН), в рамках программы
  реинтродукции амфибий.
- **Назначение:** предоставлены для исследовательской задачи повторной идентификации особей.
- **Условия:** использование в научно-исследовательских и образовательных целях. Коммерческое использование
  данных не предполагается. При использовании — ссылаться на ИПЭЭ РАН как на источник.
- **Формат публикации:** сырые фотографии публично не распространяются (предоставляются по запросу для
  исследовательских целей); в репозитории — только производные кропы каталога (`segment/crops/`),
  необходимые для запуска демонстрации.

> Персональные данные не обрабатываются: объект съёмки — животные (ФЗ-152 / GDPR неприменимы).

## Внешний набор GCN-ID (*Triturus cristatus*)

- **Источник:** Rosa et al., 2025 — GCN-ID: a dataset for individual re-identification of the great crested newt.
- **Публикация:** Zenodo, DOI [10.5281/zenodo.18000809](https://doi.org/10.5281/zenodo.18000809).
- **Лицензия:** CC-BY-4.0. В репозитории не дублируется — скачивается отдельно с Zenodo.
- **Роль в проекте:** только вспомогательная (предобучение моделей сегментации); в целевую метрику re-ID не входит.

## Предобученные модели

- **BiRefNet** (`ZhengPeng7/BiRefNet`) — сегментация; загружается с HuggingFace Hub при первом запуске.
- Внешние компоненты имеют собственные лицензии (BiRefNet — MIT; см. страницы моделей). Код проекта — под лицензией MIT (см. `LICENSE`).
